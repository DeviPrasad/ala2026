# AME 5101 - Applied Linear Algebra
Theory (LTPC: 3-0-0-3, 36 hrs)

## Course Outcomes
1. CO1 - Apply vector operations for similarity learning            -- 10 Hrs -- BL3
2. CO2 - Analyze matrix representations in AI & ML models           -- 14 Hrs -- BL4
3. CO3 - Apply matrix calculus for optimization in learning systems -- 12 Hrs -- BL3

##  Unit 1 - Vectors - 10 hrs
- Algebraic and geometric view of vectors.
- Dot product and similarity.
- Norms and regularization.
- Cosine similarity; Distance metrics.
- Standardization vs normalization.
- High-dimensional embeddings.
- Vector similarity in NLP.
- Feature representations.
- Vectorized computation.

```
Algebraic and geometric introduction to vectors.
vector addition and subtraction.
scalar-vector multiplication.
dot product.
application example from deep learning.
calculating raw scores.
norm: application example from deep learning – weight regularization.
distance, mean, standard deviation.
angle between vectors.
application example from finance – risk vs. return.
standardization vs. normalization.
application example from machine learning – tabular data vs. image data.
Euclidean distance & cosine similarity.
application examples from natural language processing & machine learning.
comparing words using embeddings.
k-means clustering algorithm.
Hadamard product.
vectorized operations: application examples from deep learning – calculating pointwise-activated scores & the softmax function.
```

## Unit 2- Matrices - 14 hrs
Matrix Representations and Transformations
- Matrix operations
- Matrix–vector product
- Matrix-matrix product
- Forward propagation in deep learning
- Batch computation
- Elementary row operations and RREF
- Solving systems of linear equations
- Linear dependence and independence of vectors.
- Rank and subspaces
- Matrix factorization
- Attention mechanism and low-rank representations.
- projections and dimensionality reduction using PCA.

```
Conceptual introduction to matrices:
    types of matrices (zero, identity, diagonal)
    matrix addition, subtraction, and transpose.
    block representation of matrices:
        application example from deep learning.
        data and weights matrices.

Extending the dot product to matrix-vector product:
    application example from deep learning – forward propagation using matrix-based approach.

Matrix-vector product as linear combination.
solving system of linear equations.
elementary row operations & RREF.
linear dependence/independence of vectors.
null space, column space, and rank of a matrix.
arthonormal basis.
projections.

Matrix-matrix product:
    application example from deep learning – batch forward propagation.
    matrix factorization.
    Eigen decomposition and SVD factorization.
    application example from machine learning – dimension reduction using PCA.
```

## Unit 3 - Matrix Calculus - 12 hrs
Matrix Calculus for Learning Systems
- introduction to sensitivity & gradient in one dimension.
- Chain rule.
- Gradient in multiple dimensions.
- Matrix derivatives.
- Optimization using gradient descent.
- Batch vs stochastic learning.
- Backpropagation for binary and multiclass classification and regression.

```
Algebraic and visual introduction to sensitivity and gradient in one dimension.
Gradient and chain rule in multiple dimensions.
Optimizing using the gradient descent method.
Batch and stochastic gradient descent.
Application example from deep learning.
backward propagation using matrix-based approach for binary and multiclass classification.
```
